# HiPNUC_Website

結合 i18n

lang/lang_pack.js 是包含所有語言文本的檔案

在 html 當中設定變數，頁面載入時會把變數內容換成當前的語言文本。

網頁載入時將自動根據瀏覽器語言設定語言，除非手動設定語言(將會記錄於 cookie)

mkdocs 每種語言為一個環境，如

mkdocs_cn\docs 底下是所有產品的資料夾

HI221、HI226 等等

md 檔擺在如 mkdocs_cn\docs\hi221\hi221um_cn.md
